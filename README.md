### Install
- To get started with the app, first clone the repo and `cd` into the directory:
  
  ```
  $ git clone https://github.com/AndresFMoya/react-calculator.git
  $ cd react-calculator
  ```
- `npm install`

- Start the app locallly:

- `npm start`
